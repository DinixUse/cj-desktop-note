# 使用方法
注意注意以下的步驟一定要跟著做一定要！\n

第一步：C盤根目錄下新建一個文件夾“note_files”。\n
第二步：在“note_files”下新建一個文本文檔命名爲“var.txt”，雙擊打開txt，然後摁一下鍵盤的“0”鍵然後保存。\n
第三步：在“note_files”下新建一個Json文件命名爲“notes.json”，雙擊選擇“用記事本打開”，然後在裏面寫入一對花括號，保存退出。\n
